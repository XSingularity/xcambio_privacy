# Privacy Policy for XCambio

**Last updated:** July 8, 2026

XCambio ("the app") is developed as a currency conversion tool for Venezuelan exchange rates. This policy explains what data the app handles.

## Data We Collect

XCambio does not collect, store, or transmit any personal data to us. The app does not have user accounts, does not use analytics, and does not display ads.

## Local Data Storage

The app stores the following data **only on your device**, using Android's local SharedPreferences:

- **Custom exchange rate**: If you set a personalized exchange rate, it is saved locally on your device so it persists between sessions. This data never leaves your device and is not accessible to us.

## Third-Party Services

To display currency exchange rates, the app fetches public data from the following third-party services:

- **Alcambio API** (`api.alcambio.app`) — provides BCV (Venezuelan Central Bank) exchange rates.
- **Binance P2P API** (`p2p.binance.com`) — provides USDT peer-to-peer exchange rates.

These requests only retrieve public rate data. The app does not send any personal or device-identifying information to these services beyond standard network request metadata (e.g., IP address), which is handled according to each provider's own privacy policy:

- Alcambio: refer to their published terms, if available.
- Binance: [https://www.binance.com/en/privacy](https://www.binance.com/en/privacy)

## Clipboard Access

The app allows you to copy amounts and rates to your device's clipboard when you tap the copy button. This is a local action performed only when you choose it and does not transmit any data externally.

## Children's Privacy

XCambio does not knowingly collect data from anyone, including children, because it does not collect data at all.

## Changes to This Policy

This policy may be updated if the app's functionality changes. Updates will be posted on this page with a revised "Last updated" date.
